# Project 6: YouTube Video Trend Analyzer

### 📝 Question:
What types of videos perform best on YouTube in terms of views, likes, and engagement?

### 🛠️ What It Does:
Analyzes YouTube video metadata to determine high-engagement categories, keywords, or creators. Displays insights through plots.

### 🎯 Goal:
To explore digital content performance using real-world social media data.

### 🧠 Concepts & Skills Learned:
* Categorical data analysis
* Bar and pie chart visualization
* Digital media analytics
* Data storytelling
